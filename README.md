# gd-wm
Wayland compositor using Godot Engine as the frontend.

Project mainly exists as an excuse to finally learn Rust.
Would not recommend using this as your primary compositor for now.

# Planned Major Features
- Simple window management with floating bordered windows
- Works both opened in another compositor and directly in a tty
- Reasonable amount of customizability

# Planned Secondary Features
- Window Physics
- Window Slicing
- Window UV Mapping
- Plugins?
